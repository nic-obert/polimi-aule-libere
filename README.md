# Polimi aule libere

Una collezione di script che, tramite scraping dei servizi online del poli, ottiene informazioni sulle aule in un dato periodo di tempo e produce un aggregato utile per capire quali e quante aule sono libere in un dato momento.

Nella directory [saved_output](saved_output) sono salvati gli ultimi file prodotti, con lo scopo di rendere rapida la consultazione di dati reali senza dover eseguire tutti gli script.
